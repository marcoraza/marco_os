# Story: Mobile Daily UX Pass

## Context
Os fluxos do dia a dia ainda tinham pequenos atritos em mobile: captura escondida, acoes com alvo de toque curto e exclusao de nota dependente de hover.

## Scope
- Expor captura rapida no menu mobile
- Melhorar tocabilidade de navegacao e notificacoes
- Tornar a exclusao de notas acessivel em touch

## Checklist

- [x] Alvos de toque melhores em navegacao e notificacoes
- [x] Exclusao de nota acessivel sem hover
